## 🔍 常见问题排查

### Q1: 安装后仍然是空白窗口

**可能原因**:
1. 旧版本缓存未清除
2. 前端资源未正确打包

**解决方案**:
```powershell
# 1. 卸载旧版本
# 控制面板 → 程序和功能 → 卸载 LSC 直播切片系统

# 2. 删除残留数据
Remove-Item "$env:APPDATA\LSC" -Recurse -Force

# 3. 重新安装新版本
# 双击新的安装包
```

### Q2: 图标显示不正确

**检查清单**:
1. 确认 `assets/icon.ico` 存在且有效
2. 检查 `electron-builder.yml` 中的 icon 配置
3. 清除缓存重新构建

**快速修复**:
```bash
cd lsc-electron
rm -rf release dist dist-electron
npx electron-builder
```

### Q3: Python后端未启动

**症状**: 
- 应用启动但无法连接后端
- 录制功能不可用

**解决方案**:
1. 确保Python 3.10+已安装
2. 检查Python是否在PATH中
3. 查看后端日志: `%APPDATA%\LSC\logs\backend.log`

**手动测试后端**:
```bash
cd python-backend
python main.py
# 应该看到 "Backend server started on ws://..."
```

### Q4: FFmpeg相关错误

**症状**:
- 录制失败
- 提示FFmpeg未找到

**解决方案**:
1. 下载FFmpeg: https://ffmpeg.org/download.html
2. 解压并将 `ffmpeg.exe` 添加到PATH
3. 或在应用设置中指定FFmpeg路径
