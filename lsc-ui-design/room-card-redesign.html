📐 设计变更说明

  • 房间选择框：预览区左上角放置自定义 checkbox，点击直接切换选中状态，无需 Ctrl 键即可多选
  • 按钮重新排列：主操作（录制/连接）占据整行，断开按钮次要位置，删除按钮最小化到角落
  • 画质选择器：放在预览区左下角，半透明背景，不遮挡画面
  • 预览控制：静音和全屏放在预览区右下角，与画质选择器同一行
  • 录制信息：时间+文件大小放右上角，始终使用暗色半透明底+白字（浅色模式不变）
  • 状态+平台：左上角合并展示，减少视觉噪音
  • 预览区高度：从 160px 提升到 180px，给画面更多空间
  • 圆角统一：卡片 14px，内部元素 6-8px

① 正常卡片
• 预览区高度 180px
• 多张卡片以 2~3列 网格排列
• 左上角 checkbox + 状态 badge
• 底部渐变栏：画质 + 静音/全屏
② 区域放大
• 预览区 填满整个房间网格区域
• 其他卡片隐藏，目标卡片独占网格
• 保留：顶部工具栏、左侧面板、右侧面板、底部控制栏
• expandedRoomId 标记当前放大的房间
③ 全屏模式
• 预览区 fixed 覆盖整个视口
• position:fixed; inset:0; z-index:9999
• ESC 键或点击关闭按钮退出全屏
• 退出后回到区域放大状态（不是正常卡片）

📐 状态机转换规则

进入流程（点击放大按钮）
正常卡片 → 点击 ⛶ → 区域放大 (expandedRoomId = roomId)
区域放大 → 点击 ⛶ → 全屏模式 (fullscreenRoomId = roomId)
退出流程（ESC / 关闭按钮）
全屏模式 → ESC → 区域放大 (fullscreenRoomId = null)
区域放大 → ESC → 正常卡片 (expandedRoomId = null)
React 状态组合
正常: expanded=null, full=null
区域放大: expanded=roomId, full=null
全屏: expanded=roomId, full=roomId
按 ESC 退出全屏
