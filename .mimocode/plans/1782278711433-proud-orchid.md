# 房间卡片控制栏重构方案（v3 — 悬浮覆盖层设计）

## 设计理念

将房间卡片的控制栏改为**预览区底部半透明悬浮覆盖层**（类似 YouTube/B站 播放器控制条），鼠标 hover 预览区时自动显示，离开后自动消失。所有播放控制（时间线 + 按钮）整合在覆盖层内。

## 修改文件清单

| 文件 | 修改内容 |
|------|---------|
| `lsc/gui/components/room_card.py` | 将 `_unified_controls` 改为注入 PreviewSurface 的 `_controls_widget`；删除 `_MiniTimeline` 独立组件 |
| `lsc/gui/pages/multi_room/page.py` | `_card_container` stretch 改回 1 |

## 详细修改

### 1. RoomCard 控制栏改为悬浮覆盖层（room_card.py）

**当前状态**：`_unified_controls`（56px高）作为独立 QWidget 放在 PreviewSurface 下方。

**目标状态**：控制栏作为 `_controls_widget` 注入 PreviewSurface，通过 hover 自动显隐。

#### 1a. `_build_ui()` 中创建控制栏

将原来的 `_unified_controls` 创建代码改为创建一个半透明覆盖层 QWidget，注入到 `self._preview_area` 的 `set_controls_widget()`：

```python
# 创建悬浮控制栏
self._overlay_controls = QWidget()
self._overlay_controls.setObjectName("roomCardOverlayControls")
oc_layout = QVBoxLayout(self._overlay_controls)
oc_layout.setContentsMargins(8, 4, 8, 4)
oc_layout.setSpacing(4)

# 时间线
self._timeline = _MiniTimeline(self._overlay_controls)
self._timeline.setObjectName("roomCardTimeline")
self._timeline.setFixedHeight(24)
self._timeline.seek_requested.connect(lambda sec: self.timeline_seek_requested.emit(self.room.room_id, sec))
oc_layout.addWidget(self._timeline)

# 按钮行
btn_row = QHBoxLayout()
btn_row.setContentsMargins(0, 0, 0, 0)
btn_row.setSpacing(4)

# 后退10s / 播放暂停 / 前进10s / 时间标签 / 预览 / 全屏 / 静音
self._back_btn = QPushButton("10s")
self._back_btn.setObjectName("roomCardOverlayBtn")
self._back_btn.setFixedWidth(40)
self._back_btn.clicked.connect(lambda: self.seek_relative_requested.emit(self.room.room_id, -10))

self._play_pause_btn = QPushButton("⏸")
self._play_pause_btn.setObjectName("roomCardOverlayBtn")
self._play_pause_btn.setFixedWidth(32)
self._play_pause_btn.clicked.connect(lambda: self.pause_clicked.emit(self.room.room_id))

self._fwd_btn = QPushButton("10s")
self._fwd_btn.setObjectName("roomCardOverlayBtn")
self._fwd_btn.setFixedWidth(40)
self._fwd_btn.clicked.connect(lambda: self.seek_relative_requested.emit(self.room.room_id, 10))

self._time_label = QLabel("00:00 / 00:00")
self._time_label.setObjectName("roomCardOverlayTimeLabel")

self._preview_btn = QPushButton("预览")
self._preview_btn.setObjectName("roomCardOverlayBtn")
self._preview_btn.clicked.connect(lambda: self.preview_clicked.emit(self.room.room_id))

self._fullscreen_btn = QPushButton("⛶")
self._fullscreen_btn.setObjectName("roomCardOverlayBtn")
self._fullscreen_btn.setFixedWidth(32)
self._fullscreen_btn.setToolTip("全屏预览")
self._fullscreen_btn.clicked.connect(lambda: self.fullscreen_clicked.emit(self.room.room_id))

self._mute_btn = _RoomCheckBox("静音")
self._mute_btn.setObjectName("roomCardOverlayCheckBox")
self._mute_btn.toggled.connect(self._on_mute_toggled)

for btn in (self._back_btn, self._play_pause_btn, self._fwd_btn, self._preview_btn, self._fullscreen_btn):
    btn.setCursor(Qt.CursorShape.PointingHandCursor)

btn_row.addWidget(self._back_btn)
btn_row.addWidget(self._play_pause_btn)
btn_row.addWidget(self._fwd_btn)
btn_row.addWidget(self._time_label, 1)
btn_row.addWidget(self._preview_btn)
btn_row.addWidget(self._fullscreen_btn)
btn_row.addWidget(self._mute_btn)
oc_layout.addLayout(btn_row)

# 注入到 PreviewSurface 的 hover 覆盖层
self._preview_area.set_controls_widget(self._overlay_controls)
```

**不再需要**：
- `_unified_controls` 及其 `setFixedHeight(56)` 和 `inner.addWidget()`
- `_PreviewCornerButton`（全屏按钮已整合到覆盖层按钮行）

#### 1b. 主题样式（theme.py）

添加半透明覆盖层样式：

#### 1c. refresh() 方法

控制栏状态管理不变（`_preview_btn` / `_play_pause_btn` 显隐切换、`_time_label` 更新）。

#### 1d. set_timeline_data() 方法

更新 `_time_label` 和 `_timeline` 不变。

### 2. 卡片区域 stretch 恢复（page.py）

将 `left_layout.addWidget(self._card_container, 0)` 改回 `left_layout.addWidget(self._card_container, 1)`，让卡片区域占满可用空间。保持 `_update_card_container_min_height()` 调用不变。

### 3. PreviewSurface 不需修改

`preview_surface.py` 已有完整的 hover 显隐 `_controls_widget` 机制：
- `enterEvent` → `_set_overlay_expanded(True)` → 控制条从底部滑出
- `leaveEvent` → `_set_overlay_expanded(False)` → 控制条滑出视口外
- `_update_overlay_geometry()` 中角标按钮始终可见，控制条跟随 hover

## 关键设计约束

**不破坏的功能**：
1. 预览播放（mpv_widget.py 的 rebind/deferred_init 链路）
2. 导出/切片（`room.record_output_path` 路径）
3. 总时间线区控制栏（control_bar.py 独立于卡片）
4. 右侧录制信息面板（page.py 的 DetailPanel 刷新）
5. 房间增删/选择/多选逻辑

**修改范围限制**：
- 只改 room_card.py 的控制栏区域和 theme.py 的样式
- page.py 仅改 stretch 因子
- 不动 mpv_widget.py、control_bar.py、timeline.py、preview_surface.py

## 验证方案

1. 启动程序，添加 2-3 个房间
2. 鼠标 hover 到预览区 → 底部出现半透明控制栏（时间线 + 按钮）
3. 鼠标移出预览区 → 控制栏自动消失
4. 点击"预览" → 视频播放，按钮变为暂停图标
5. 点击后退/前进 10s → 视频跳转
6. 点击全屏 → 进入全屏预览
7. 鼠标 hover 到不同房间的预览区 → 各自独立显示控制栏
8. 卡片区域高度随卡片数量自动调整
9. 总时间线区按钮在有录制视频时可用
10. 右侧面板录制信息有数据
