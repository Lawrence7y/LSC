# 计划：设置右侧部分为不可拉伸状态

## 需求
将多直播间页面的右侧部分设置为不可拉伸状态。

## 当前实现分析

当前 `_splitter` 配置：
```python
self._splitter = QSplitter(Qt.Horizontal, self._page_body)
self._splitter.setHandleWidth(8)
self._splitter.setChildrenCollapsible(False)
```

右侧 `right_scroll` 配置：
```python
right_scroll.setMinimumWidth(360)
right_scroll.setMaximumWidth(400)
right_scroll.setSizePolicy(QSizePolicy.Fixed, QSizePolicy.Expanding)
```

问题：虽然设置了 `QSizePolicy.Fixed`，但 `QSplitter` 仍然允许拖拽调整大小。

## 解决方案

使用 `QSplitter.setStretchFactor` 设置拉伸因子，让左侧占据所有多余空间，右侧保持固定宽度。

```python
# 左侧拉伸因子为1（占据所有多余空间）
self._splitter.addWidget(left_widget)
# 右侧拉伸因子为0（不拉伸）
self._splitter.addWidget(right_scroll)
self._splitter.setStretchFactor(0, 1)  # 左侧
self._splitter.setStretchFactor(1, 0)  # 右侧
```
