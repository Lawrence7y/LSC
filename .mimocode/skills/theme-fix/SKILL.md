# LSC UI主题修复技能

自动化检查和修复LSC项目的UI主题问题，确保深浅主题下的一致性。

## 使用方法

```bash
# 检查主题一致性
python -m mimocode skill run theme-fix --check

# 修复主题问题
python -m mimocode skill run theme-fix --fix

# 生成主题预览
python -m mimocode skill run theme-fix --preview
```

## 检查项目

### 1. 颜色一致性
- 主题token定义完整性
- 组件颜色引用正确性
- 深浅主题颜色对比度
- 强调色使用一致性

### 2. 样式一致性
- 字体大小统一
- 间距规范一致
- 圆角大小统一
- 阴影效果一致

### 3. 交互一致性
- 悬停效果
- 点击反馈
- 禁用状态
- 选中状态

### 4. 布局一致性
- 组件对齐
- 间距规范
- 响应式布局
- 高DPI支持

## 修复流程

1. **问题扫描** - 自动扫描所有UI组件
2. **问题分类** - 按严重程度分类
3. **生成修复** - 自动生成修复代码
4. **验证修复** - 运行程序验证修复效果
5. **生成报告** - 输出修复报告

## 常见问题

### 1. 主题切换不生效
- 检查`generate_stylesheet()`调用
- 确保`dark`参数正确传递
- 检查内联样式覆盖

### 2. 颜色不一致
- 检查ThemeColors定义
- 确保使用token而非硬编码
- 检查dim变体定义

### 3. 布局错乱
- 检查QSS选择器
- 确保组件ID正确
- 检查布局嵌套

## 注意事项

- 修复前备份相关文件
- 使用主题token而非硬编码颜色
- 确保`generate_stylesheet()`正确调用
- 测试深浅主题切换效果
- 检查高DPI显示效果
