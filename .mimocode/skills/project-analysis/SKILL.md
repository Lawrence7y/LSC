# LSC 项目分析技能

自动化分析LSC项目的架构、代码质量和依赖关系，生成详细的项目分析报告。

## 使用方法

```bash
# 全面分析
python -m mimocode skill run project-analysis --full

# 架构分析
python -m mimocode skill run project-analysis --architecture

# 代码质量分析
python -m mimocode skill run project-analysis --quality

# 依赖分析
python -m mimocode skill run project-analysis --dependencies
```

## 分析维度

### 1. 架构分析
- 项目结构
- 模块划分
- 依赖关系
- 设计模式
- 入口点分析

### 2. 代码质量
- 代码复杂度
- 重复代码
- 类型注解
- 异常处理
- 日志记录

### 3. 测试覆盖
- 测试覆盖率
- 测试质量
- 测试完整性
- 边界测试
- 性能测试

### 4. 依赖分析
- 外部依赖
- 版本兼容性
- 安全漏洞
- 许可证合规
- 依赖树分析

### 5. 性能分析
- 启动性能
- 运行时性能
- 内存使用
- CPU使用
- 磁盘IO

## 分析流程

1. **数据收集** - 收集项目信息
2. **静态分析** - 分析代码结构
3. **动态分析** - 运行测试和性能分析
4. **问题识别** - 发现关键问题
5. **生成报告** - 输出分析报告

## 注意事项

- 分析前确保代码可以编译
- 运行测试确保现有功能正常
- 记录具体的问题位置和修复建议
- 考虑项目的实际使用场景
- 优先处理高影响问题
