# Task 11: 完整编译和功能验证

## 任务目标

编译完整项目，启动 Shotcut，验证所有新功能正常工作。

## 执行步骤

### Step 1: 完整编译

```powershell
cd D:\Project\直播切片\shotcut-source\build

# 清理并重新配置
cmake .. -G "Visual Studio 17 2022" -A x64 -DCMAKE_TOOLCHAIN_FILE=C:/vcpkg/scripts/buildsystems/vcpkg.cmake

# 编译
cmake --build . --config Release
```

**预期**：编译成功，0 错误。

### Step 2: 启动验证

```powershell
.\Release\shotcut.exe
```

**检查项**：

| 检查项 | 预期行为 |
|--------|----------|
| Shotcut 正常启动 | 主界面加载，无崩溃 |
| 直播源面板可见 | 左侧区域有「直播源」标签 |
| AI分析面板可见 | 右侧区域有「AI分析」标签 |
| 时间线正常 | 原有编辑功能正常 |
| 滤镜面板正常 | 原有特效功能正常 |

### Step 3: 直播录制测试

```
1. 在直播源面板输入URL（或使用本地测试流）
2. 点击「开始录制」
3. 观察状态变化：就绪 → 录制中
4. 检查时长和文件大小更新
5. 点击「停止」
```

**注意**：需要系统安装 FFmpeg 且可在 PATH 中找到。

### Step 4: AI分析测试

```
1. 加载一个已录制的视频
2. 切换到 AI分析面板
3. 点击「开始分析」
4. 观察进度条更新
5. 验证精彩片段表格有数据显示
6. 双击某行验证是否弹出导出信号
```

### Step 5: 异常处理测试

```
1. 输入无效URL → 应显示错误
2. 录制中停止FFmpeg进程 → 应自动重连
3. 磁盘空间不足时录制 → 应提示错误
```
