# 已归档：旧版 VSCode 风格 UI 原型

**归档日期**：2026-06-21
**原路径**：`docs/ui-prototype/index.html`

## 为什么归档

这份原型描绘的是一个"专业非线性剪辑软件"形态——多轨道时间线（视频/音频/字幕）、媒体库、菜单栏、剃刀工具、撤销重做、字幕编辑、BGM 轨道。

实际项目的产品定位是**直播录制 + 快速切片工具**，并不实现上述专业剪辑能力。保留这份原型会：

- 让新加入的设计师 / 产品 / 开发对项目方向产生严重误解
- 给用户制造"专业剪辑软件"的虚假期待
- 与实际采用的橙色卡片式设计（见 `docs/superpowers/prototypes/ui-prototype.html`）冲突
